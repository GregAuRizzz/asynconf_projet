# Green Bank Calculator

Ce projet est une application web qui permet aux utilisateurs de calculer leur empreinte écologique 
en fonction de divers critères liés à leur véhicule et à leurs habitudes de conduite. Il utilise des 
données sur différents types de véhicules, années de fabrication, types d'énergie, kilométrages, etc.

## Fonctionnalités

- Sélection du type de véhicule
- Choix de l'année de fabrication
- Sélection du type d'énergie
- Indication du kilométrage annuel
- Sélection du nombre de passagers

En fonction des choix de l'utilisateur, l'application fournit un taux d'emprunt.

## Technologies utilisées

HTML/CSS/Js

## Données

Les données utilisées pour le calcul de l'empreinte écologique sont stockées dans un fichier JSON (`donnes.json`) 
qui contient des informations sur différents types de véhicules, années de fabrication, énergies, kilométrages, taux, et 
le nombre de passagers. J'ai utilisé un json pour la facilité de modification du projet
